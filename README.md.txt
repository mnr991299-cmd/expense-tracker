## Description
This is a Python-based Expense Tracker application that allows users to manage their expenses efficiently. Users can:  
- Add expenses with date, category, amount, and description  
- View all expenses in a neat table format  
- Calculate total expenses  
- Filter expenses by category or month  
- Sort expenses by date or amount  
- Store expenses permanently in a file (`expenses.txt`)
